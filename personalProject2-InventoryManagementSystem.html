Project Description : Inventory Management System

Developed a Python-based Inventory Management System to help users, businesses, or organisations manage inventory efficiently. It allows users to add and update product and supplier information, track stock levels, place restocking orders, and generate reports. This program streamlines the inventory management process by providing essential features that support better stock control and accurate record-keeping.

Key Features:

  • Add and update product details (with duplicate ID validation)
  • Manage supplier information and contact details
  • Place orders to restock low quantity products
  • View inventory levels of the products stored in the system, sorted by quantity (lowest to highest)
  • Generate Reports

Key Functions in My Inventory Management System

1. Product Related Functions

This function allow users to add a new product into the inventory management system and update existing product details.

2. Supplier Related Functions

This function allow users to add a new supplier into the system and update supplier information.

3. View Report Functions

This function allows users to generate a report to view the inventory levels of products stored in the system, such as high-stock and low-stock items. This helps users easily identify which products are running low, so they can plan for restocking.

4. Main Program Function

This function allows users to run the full inventory management system program, where all core functions such as product/supplier management, placing orders, and generating inventory reports are handled from a single interface.
