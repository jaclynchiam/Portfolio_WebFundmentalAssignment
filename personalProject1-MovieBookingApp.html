Project: RoyalScreen – Movie Booking Application

  • Developed RoyalScreen, a movie booking application, from the ground up as part of my Foundation Year Introduction to Programming course.
  • Designed the UI/UX using Canva, focusing on creating an intuitive and user-friendly interface to enhance the overall movie booking experience.
  • Developed the full front-end using JavaScript, optimising the ticket booking process for better convenience and efficiency.
  • I was in charge of coding the mock-up design of the Movie List Page using JavaScript for our proposed mobile application, and responsible for the UI/UX design of the Refund Tickets Page and User Profile Page.
Built with: JavaScript Canva

Project Description

As part of my Foundation Year Introduction to Programming assignment, my group and I have designed and developed a movie booking application called RoyalScreen using Canva for the UI/UX design and JavaScript for front-end development. We started by reviewing several existing movie booking platforms and found some hidden issues and weak points in their user experience. Based on these findings, we decided to build our own movie booking application to solve those problems and create a smoother, more user-friendly booking experience for users.
