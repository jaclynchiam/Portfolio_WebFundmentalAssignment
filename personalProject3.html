Java GUI Mini Projects: Coffee Shop Ordering System and Swimming Competition Timer

These projects were developed in August 2023 as part of individual assignments during my Foundation in Arts. Both are Java GUI programs using JOptionPane, designed to handle user input, perform calculations, and display results interactively.

☕ Coffee Shop Ordering System (Java GUI)

Project Description

This is a Java-based GUI application that simulates a coffee shop ordering system. Customers can select from a list of beverages and optional extra items, and the program will calculate and display the total cost. The interface was built using JOptionPane for a simple and interactive user experience.

Features
Choose from 3 beverages:

  • Regular Coffee - RM5.50
  • Cappuccino - RM7.49
  • Latte - RM8.00

Add-ons available

  • Whipped Cream - RM1.25
  • Chocolate Syrup - RM2.00
  • Final total cost displayed via dialog box

Language Used:

  • Java
  • Swing (JOptionPane dialogs)

🏊‍♀️ Swimming Competition Ranking System (Java GUI)

Project Description

This is a Java-based GUI application designed for a local swimming competition. It uses JOptionPane dialogs to take input for the names and completion times (in seconds) of three swimmers participating in a 100-meter freestyle event. After collecting the data, the program compares the times and displays the swimmers' rankings from fastest to slowest in a pop-up message box.

Features

  • Input swimmer names and completion times using dialog box
  • Automatically compare and rank swimmers based on their race time
  • Display results in order from faster to slowest
  • User-friendly interface using Java's JOptionPane Dialogs

Language Used:

  • Java
